## 任务

我正在做一个 rust 的 rpc 框架，它支持：

- client/server 异步双向通信，包含：
  - 信号控制（request、response、event）
  - 实时数据传输（stream，需要支持时间同步）
- 局域网零配置服务发现（可选能力，支持手动指定）

目标场景是用于 client 和 server 之间的实时通信，

方便开发者快速注册实现自定义的命令处理、事件监听和数据流的实时传输。
